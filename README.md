# 🛍️ Kixikila Market — E-Commerce Front-End

Projecto de loja online de produtos angolanos, desenvolvido com HTML, CSS e JavaScript puro (sem frameworks, sem TypeScript).

---

## 📁 Estrutura do Projecto

```
kixikila-market/
│
├── index.html          → Página principal (estrutura HTML)
│
├── css/
│   └── style.css       → Todos os estilos (variáveis, layout, responsivo)
│
└── js/
    ├── data.js         → Dados: categorias e produtos angolanos
    └── app.js          → Lógica: carrinho, filtros, pesquisa, modal
```

---

## 🚀 Como Abrir no VS Code

1. Abre a pasta `kixikila-market` no VS Code
2. Instala a extensão **Live Server** (se ainda não tiveres)
3. Clica com o botão direito em `index.html` → **"Open with Live Server"**
4. O browser abre automaticamente em `http://127.0.0.1:5500`

> Alternativa: abre directamente o ficheiro `index.html` no browser (duplo clique).

## ✨ Funcionalidades

| Funcionalidade      | Descrição                                              |
|---------------------|--------------------------------------------------------|
| 🏪 Catálogo         | 20 produtos angolanos com preços em Kwanza (Kz)        |
| 🔍 Pesquisa         | Pesquisa em tempo real por nome, descrição ou origem   |
| 🗂️ Categorias       | 8 categorias filtráveis com contagem de produtos       |
| 🏷️ Filtros          | Filtrar por Promoções, Novidades e Destaques           |
| ↕️ Ordenação        | Ordenar por preço (↑↓) ou nome (A→Z)                 |
| 🛒 Carrinho         | Adicionar, remover, alterar quantidade, total em Kz    |
| ❤️ Wishlist         | Lista de desejos com contador no header                |
| 🔍 Modal            | Detalhe completo do produto ao clicar                  |
| 📋 Código Promo     | Copiar código ANGOLA15 com um clique                   |
| 📱 Responsivo       | Funciona em desktop, tablet e telemóvel                |
| 🎨 Animações        | Fade-in, hover, float e transições suaves              |
