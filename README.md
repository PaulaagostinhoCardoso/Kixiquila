# 👑 Kixikila Market — Angola

A plataforma de e-commerce premium que celebra o melhor de Angola.

## 🚀 Sobre o Projecto
O **Kixikila Market** é uma solução Full-Stack moderna para o comércio de produtos angolanos. Com um design focado no luxo e na experiência do utilizador, a plataforma oferece uma montra sofisticada e ferramentas administrativas robustas.

## ✨ Funcionalidades em Destaque
- **UI de Luxo**: Tema Dark Mode com acentos Pink e Gold.
- **Gestão Total**: Painel administrativo para CRUD de produtos e categorias.
- **Performance**: Backend leve em PHP com integração MySQL optimizada.
- **Responsivo**: Adaptado para qualquer dispositivo.
